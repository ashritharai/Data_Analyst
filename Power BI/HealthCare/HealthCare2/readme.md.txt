All India Institutes of Medical Sciences Dataset
Data Source:
All India Institutes of Medical Sciences Wikipedia page
Link: https://en.wikipedia.org/wiki/All_India_Institutes_of_Medical_Sciences

Steps to import data in Power BI:
1. Get Data in the 'Data' group of the Home tab.
2. Select Web from the listed sources and enter the URL
3. The web page could contain multiple tabls. Select the table you want to analyze.
4. Load or Tranform data if required.

The visualization provides the following insights:
1. Number of AIIMS colleges established by Prime Ministers
2. Number of AIIMS in different states
3. Functional status of different colleges
